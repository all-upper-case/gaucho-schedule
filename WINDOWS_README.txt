GAUCHO SCHEDULE - PORTABLE WINDOWS EDITION

1. Keep the entire GauchoSchedule folder together.
2. Double-click GauchoSchedule.exe.
3. Your browser will open the schedule automatically.
4. Leave the small server window open while using the schedule.
5. Close that window when you are finished.

No Python installation is needed.

Your live schedule database is stored in:
%LOCALAPPDATA%\GauchoSchedule\gaucho_schedule.sqlite3

Use the Backup link inside the app regularly. A downloaded .sqlite3 backup
contains employees, schedules, versions, settings, and imported history.

The app runs only on this computer at http://127.0.0.1:5000. It does not need
internet access and should not be exposed publicly.
